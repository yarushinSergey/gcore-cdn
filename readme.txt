Gcore Plugin

== Description ==

Increase customer loyalty and the time spent on your website with Gcore Content Delivery Network.
This plugin allows you to easily enable Gcore CDN on your WordPress and accelerate website loading in any country, handling any load on your service.

= How does it work? =
It takes about 15â€¯minutes to set up, just enter personal domain from Gcore control panel.
The plugin will automatically configure your website to replace existing static content links with the CDN links, efficiently speeding up content delivery.

= Plugin features =
* Replace static links with CDN links
* Specify file extensions
* Include specific directories
* Set exceptions

= How to connect CDN =

1. Sign up for free at <a href="https://gcore.com/cdn/" target="_blank">https://gcore.com/cdn/</a>.
2. Create and configure a CDN resource.
3. Install the plugin on your website.
4. Specify a personal domain from your Gcore control panel in the plugin settings.
5. Select the types of files and folders you want to distribute via CDN.
For quick CDN setup refer to our <a href="https://support.gcore.com/hc/en-us/categories/115000655829" target="_blank">knowledge base</a> or contact our friendly <a href="mailto:support@gcore.com">technical support</a>.
